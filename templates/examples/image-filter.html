How Image Filters Work

Fragment shaders process pixels in parallel on the GPU:

  • Point Operations: Modify each pixel independently (brightness, invert)
  • Convolution: Use neighboring pixels (blur, sharpen, edge detect)
  • Color Space: Convert between RGB, HSL, etc.
  • Performance: GPU processes millions of pixels simultaneously
