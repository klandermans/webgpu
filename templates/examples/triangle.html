🔺 Basic Triangle

This example demonstrates the fundamentals of WebGPU rendering:

  • Vertex Buffer: Stores vertex positions and colors
  • Vertex Shader: Processes each vertex position
  • Fragment Shader: Determines pixel colors
  • Pipeline: Configures how GPU renders graphics
FPS: 0 Vertices: 3

How It Works

  1. WebGPU device and context are initialized
  2. Vertex data (positions & colors) is uploaded to GPU buffer
  3. WGSL shaders define vertex transformation and coloring
  4. Render pipeline connects shaders to output
  5. Command encoder creates draw commands
  6. GPU executes commands and renders to canvas
