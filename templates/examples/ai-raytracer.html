🧠 AI Analysis

Start the webcam and click "Analyze Scene" to detect objects and generate a ray traced scene powered by AI.

The system will:

  • Detect objects using computer vision
  • Use LLM to describe the scene
  • Generate 3D objects matching detected items
  • Ray trace the scene in real-time with WebGPU

📐 Generated Scene

    ⚙️ How It Works

    1. Computer Vision: TensorFlow.js COCO-SSD detects objects in your webcam feed
    2. LLM Integration: WebLLM (Qwen) generates a scene description based on detected objects
    3. Scene Generation: Objects are converted to 3D spheres with appropriate materials
    4. Ray Tracing: WebGPU compute shaders trace millions of rays per frame
    5. Real-time Rendering: Reflections, shadows, and global illumination updated interactively

    🎯 Controls

    • Mouse Drag: Rotate camera
    • Scroll: Zoom in/out
    • Ray Bounces: Control reflection depth
    • Resolution: Balance quality vs performance
