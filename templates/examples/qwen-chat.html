How It Works

This chat runs Qwen LLM entirely in your browser using WebLLM and WebGPU:

  • Model Download: Quantized model weights are downloaded once and cached
  • WebGPU Compute: Matrix operations run on your GPU for fast inference
  • Privacy: All computation happens locally - no data sent to servers
  • Streaming: Responses appear token by token as they're generated

System Requirements

  • Browser: Chrome 113+ or Edge 113+ with WebGPU support
  • GPU: Dedicated GPU recommended (4GB+ VRAM for larger models)
  • RAM: 8GB+ system RAM recommended
  • Storage: 1-4GB for model cache (downloaded once)
