How LLM Inference Works on GPU

  1. Tokenization: Input text is converted to token IDs
  2. Embedding: Tokens are mapped to dense vectors
  3. Transformer Layers: Each layer performs:
    • Self-Attention: Compute attention scores between all tokens
    • Feed-Forward: Two matrix multiplications with activation
    • LayerNorm & Residual connections
  4. Output Head: Final projection to vocabulary size
  5. Sampling: Select next token (greedy, top-k, or nucleus)

Why GPU? A single forward pass through a 7B parameter model requires billions of operations. GPU parallelism makes this feasible in real-time.
